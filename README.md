## Getting Started

### Requirements

- [Yarn](https://yarnpkg.com/en/)
- [Next.js](https://nextjs.org/)

### Quick Start

Before you do this you must clone this repo.

Then open your directory in your favorite terminal and run this

```
yarn install
```

Then Run next project:

```
yarn run dev
```
